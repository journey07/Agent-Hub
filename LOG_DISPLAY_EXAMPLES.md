## 🎨 스타일 정보

- **사용자명 색상:** `#94a3b8` (slate-400)
- **위치:** 액션 메시지 바로 뒤, `marginLeft: '6px'`
- **형식:** `- {userName}`

- **사용자명 색상:** `#64748b` (slate-500)
- **위치:** 액션 메시지 바로 뒤, `marginLeft: '8px'`

## ✅ 확인 사항

1. ✅ 사용자명이 있는 로그만 표시 (null 체크)
2. ✅ 시스템 로그(Heartbeat 등)는 사용자명 없음 (정상)
3. ✅ 로그인 로그도 사용자명 표시
4. ✅ 모든 API 호출 로그에 사용자명 표시
5. ✅ 대시보드와 디테일 페이지 모두 일관된 형식
